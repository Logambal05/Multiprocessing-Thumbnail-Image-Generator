# Thumbnail Generator with Multiprocessing

A Python project to generate high-resolution thumbnails from images using a **multiprocessing producer-consumer approach**.  
The project also includes a **Streamlit app** for easy upload and visualization.

---

## Features

- Generate thumbnails for uploaded images.
- Use pre-existing sample images from the `assets/` folder.
- High-resolution thumbnails with clear quality.
- Multiprocessing ensures faster processing for multiple images.
- Download generated thumbnails individually.
